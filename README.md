# ✦ About

VERONESI is a cinematic fashion and creative direction portfolio inspired by music, emotional atmosphere and visual storytelling.

This project merges:
- Front-end development
- Musical identity
- Editorial aesthetics
- Conceptual fashion
- Interactive experiences
- Dark luxury visual identity

Built with React, Vite, TailwindCSS and Framer Motion.
